Searches originating from this page will look for items at the Martin Luther College Library in New Ulm, MN.

The MLC Library is part of the Traverse des Sioux Library Cooperative (TdS).

Items not available at the MLC library can quickly and easily be borrowed from other TdS libraries by using an item's "Request It" button.

If you need assistance with the catalog or locating library materials, please ask library staff.
